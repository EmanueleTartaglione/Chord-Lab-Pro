# Music Theory Pro

A comprehensive, interactive music theory web app built with Streamlit. Explore chords, scales, progressions, and music theory — all in one place.

**Live app:** [music-theory-pro.streamlit.app](https://chord-lab-pro.streamlit.app)

## Features

- **Chord Lab** — Interactive piano (2 octaves) and guitar fretboard (14 frets) with 40 chord qualities, real-time chord detection, and smart guitar voicing
- **Scale Explorer** — 30+ scales across 6 categories (Diatonic Modes, Minor Variants, Pentatonic & Blues, Symmetric, World & Ethnic, Jazz) with piano/guitar visualization, diatonic chords, staff notation, and theory/history for each scale
- **Chord Theory** — 9 educational sections covering intervals, triads, 7th chords, suspended, added tones, extended, altered, inversions, and power chords
- **Chord Progressions** — 9 genres (Pop, Jazz, Bossa Nova, Flamenco, Blues, Rock, Classical, R&B/Neo-Soul, Country) with key transposition and playback
- **Circle of Fifths** — Interactive SVG visualization with key signatures, diatonic chords, and modulation paths

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
